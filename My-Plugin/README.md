## 简介
让代码越写越少，积累工具，起步阶段：

## 包含组件
| module             |    introduce
| -------------------|----------------------------------------------------------------------------------
| MyExcel            |     使用xml完全配置化Excel、Csv表格的模板生成，数据读取，数据导出
| MyGenerator        |     MyBatis-Plus代码生成
| MyUtil             |     常用工具

## 安装
### Maven
在项目的pom.xml的dependencies中加入以下内容:

```xml
<dependency>
    <groupId>cn.renlm.plugins</groupId>
    <artifactId>My-Plugin</artifactId>
    <version>1.0.1-SNAPSHOT</version>
</dependency>
```
